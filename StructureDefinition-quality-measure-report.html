Resource Profile: Quality Measure Report

Official URL: http://ibm.com/fhir/cdm/StructureDefinition/quality-measure-report Version: 7.0.0
Active as of 2022-09-06 Computable Name: QualityMeasureReport

An abstract quality measure report profile. This profile serves as the base for organizational, practitioner, and patient-focused quality measure reports.

Usage:

  • Derived from this Resource Profile: Organization Quality Measure Report, Patient Quality Measure Report, Practitioner Quality Measure Report
