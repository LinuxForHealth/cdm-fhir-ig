Data Type Profile: Data lineage meta with process extensions

Defining URL:http://alvearie.io/fhir/StructureDefinition/process-meta
Version:0.1.0
Name:ProcessMeta
Title:Data lineage meta with process extensions
Status:Active as of 2021-04-14T11:21:37-05:00
Definition:

Metadata about a resource. The ProcessMeta StructureDefinition augments the base FHIR Meta profile with a set of extensions to support data lineage.

Source Resource:XML / JSON / Turtle

The official URL for this profile is: 

http://alvearie.io/fhir/StructureDefinition/process-meta
