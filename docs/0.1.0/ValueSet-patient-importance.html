ValueSet: Patient Importance Value Set

Summary

Defining URL:http://ibm.com/fhir/cdm/ValueSet/patient-importance
Version:0.1.0
Name:PatientImportanceValueSet
Title:Patient Importance Value Set
Status:Active as of 2021-06-14T16:26:11-05:00
Definition:

Patient importance status codes derived from customer-specific code mappings, used to trigger or limit IBM Health Data Connect patient operations.

Source Resource:XML / JSON / Turtle

References

  • CDM Patient

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include all codes defined in http://terminology.hl7.org/ValueSet/v3-PatientImportance
  • Include all codes defined in http://ibm.com/fhir/cdm/CodeSystem/patient-importance
