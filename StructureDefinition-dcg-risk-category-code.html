Extension: Risk Category Code

The IBM Advantage Suite standard code for the risk category, using values of Risk Score Concurrent NonRescale. The risk category only applies to the Commercial All-encounter Medical and Drug models.

The official URL for this extension is: 

http://ibm.com/fhir/cdm/StructureDefinition/dcg-risk-category-code
